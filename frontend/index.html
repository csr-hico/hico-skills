HICOSkill Library

Skill & Agent Library

Durchsuchbare Bibliothek von KI-Skills und Agents, bereitgestellt als Remote-MCP-Server. Binde deinen LLM-Client ein - er greift dann selbstständig auf die passenden Skills zu.

MCP-Server einbinden

Claude
  • claude.ai: Settings -> Connectors -> Add custom connector, URL {{MCP_URL}}, dann Authentik-Login.
  • Claude Code: claude mcp add --transport http hico-skills {{MCP_URL}}
OpenAI
  • ChatGPT (Developer mode / Connectors): MCP-Server-URL {{MCP_URL}} hinzufügen.
  • Bei der Authentifizierung „Dynamic Client Registration" (DCR) / „dynamisch" wählen - nicht CIMD. Der Server registriert den Client automatisch; du musst keine Endpunkte oder Metadaten-URL eintragen.
  • Responses API: tools:[{type:"mcp", server_url:"{{MCP_URL}}"}] mit Authorization.
Gemini
  • Gemini CLI: in ~/.gemini/settings.json unter mcpServers mit httpUrl: "{{MCP_URL}}".
  • Gemini SDK: MCP-Server über den SDK-MCP-Support einbinden.

Verbindungsdaten
