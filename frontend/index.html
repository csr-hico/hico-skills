Skill & Agent Library

A searchable library of AI skills and agents, served as a remote MCP server. Connect your LLM client and it will reach for the right skills on its own.

Connect the MCP server

Claude
  • claude.ai: Settings -> Connectors -> Add custom connector, URL {{MCP_URL}}, then sign in with Authentik.
  • Claude Code: claude mcp add --transport http hico-skills {{MCP_URL}}
OpenAI
  • ChatGPT (Developer mode / Connectors): add the MCP server URL {{MCP_URL}} and sign in - no advanced OAuth settings needed.
  • Responses API: tools:[{type:"mcp", server_url:"{{MCP_URL}}"}] with Authorization.
Gemini
  • Gemini CLI: in ~/.gemini/settings.json under mcpServers with httpUrl: "{{MCP_URL}}".
  • Gemini SDK: add the MCP server via the SDK's MCP support.
